Projet : Plan Mobilité pour les Salariés

Dans le cadre d'un projet pour une entreprise souhaitant développer un plan de mobilité pour ses salariés, j'ai pris part à la création d'un site web statique. Le projet s'appuie sur une analyse approfondie des besoins des salariés afin de développer des fonctionnalités essentielles visant à améliorer leur utilisation du site.

Projet SQL : Analyse des Jeux Olympiques

Dans le cadre d'un projet SQL passionnant, j'ai eu l'opportunité de travailler sur une base de données des Jeux Olympiques. Mon rôle consistait à aérer, manipuler et extraire des données significatives de cette base afin de dresser une étude statistique approfondie. À travers des requêtes SQL complexes, j'ai analysé les performances des athlètes, les tendances au fil des années, et j'ai contribué à générer des rapports détaillés.

Projet Java : Chasse au monstre

Le but de ce projet était de mettre en application le modèle MVC (Model-View-Controller) pour développer un jeu de chasse à 2 joueurs, le monstre souhaitant sortir du labyrinthe et laissant une trace de son passage sur chaque case, et le chasseur qui doit traquer le monstre en repérant les traces de celui-ci. Durant ce projet, j'ai pu élaborer des algorythmes complexes de recherche de chemin afin de mettre sur pied un ordinateur capable de prendre des décisions à la place du joueur. J'ai également appris à faire la liaison entre l'interface de jeu, le stockage des données et le traitement des informations sur des couches de codes bien distinctes en utilisant le principe du Single Responsability.

Projet JavaScript : Jeu "Shoot'em up" multijoueur

Le but de ce premier projet JavaScript était de réaliser un jeu multijoueur qui met en place des échanges entre un serveur web et un affichage client sur le navigateur web du joueur. Le jeu permet la présence de plusieurs joueurs sur une même partie et effectue une sauvegarde des meilleurs scores réalisés sur le serveur.
